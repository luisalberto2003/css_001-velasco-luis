Guía de Propiedades CSS

Este manual muestra las propiedades CSS más importantes. Observa cómo este mismo contenido HTML cambia de apariencia según el archivo CSS que uses. En la línea 8 puedes cambiar entre tema1.css (azul), tema2.css (oscuro), tema3.css (colorido) o tema4.css (gris).

¿Qué puedes cambiar con CSS?

Cada clase HTML (header, hero, article, card, etc.) puede tener su propio estilo CSS independiente. Los archivos tema1.css a tema4.css demuestran cómo diferentes colores y propiedades transforman la misma estructura.

Color (background & color)

background: cambia el color de fondo del elemento. Ejemplos:

background: #4a90e2; → fondo azul
background: #2d2d2d; → fondo gris oscuro
background: white; → fondo blanco

color: cambia el color del texto. Ejemplos:

color: white; → texto blanco
color: #333; → texto gris oscuro
color: #4a90e2; → texto azul

Espaciado (padding & margin)

padding: espacio interno dentro del elemento. Ejemplos:

padding: 20px; → 20px en todos los lados
padding: 10px 20px; → 10px arriba/abajo, 20px izquierda/derecha

margin: espacio externo alrededor del elemento. Ejemplos:

margin: 20px; → 20px alrededor de todo el elemento
margin: 0; → sin margen

Bordes y estructura

border: crea bordes alrededor de elementos. Ejemplos:

border: 2px solid #4a90e2; → borde de 2px, sólido, azul
border-top: 5px solid #e74c3c; → solo borde superior

text-align: alinea el texto. Valores:

text-align: center; → centrado
text-align: left; → izquierda (por defecto)
text-align: right; → derecha

Display y visibilidad

display: controla cómo se muestra el elemento. Ejemplos:

display: block; → ocupa toda la línea (como div, p)
display: inline; → no rompe línea (como span, a)
display: inline-block; → combina ambos
display: none; → oculta el elemento completamente

Otras propiedades útiles

font-family: cambia la fuente del texto.

font-family: Arial, sans-serif; → fuente Arial
font-family: Georgia, serif; → fuente Georgia

font-weight: grosor del texto.

font-weight: bold; → texto en negrita
font-weight: normal; → texto normal

Cómo funciona CSS

Selector: indica qué elemento HTML afectar.

body { } → afecta a todo el body
.hero { } → afecta a elementos con class="hero"
header { } → afecta a todos los header

Propiedades: dentro de las llaves defines los estilos.

.hero { background: blue; color: white; }

Tema 1 (Azul)

En tema1.css: colores azules (#4a90e2), fondo claro (#f0f8ff), bordes azules.

Ejemplo de código:

.hero { background: white; padding: 30px; }

Tema 2 (Oscuro)

En tema2.css: fondo oscuro (#1a1a1a), acentos verdes (#00ff88), texto claro.

Ejemplo de código:

.hero { background: #2d2d2d; border: 3px solid #00ff88; }

Tema 3 (Colorido)

En tema3.css: colores vibrantes (rojo #ff6b6b, turquesa #4ecdc4, amarillo #ffe66d).

Ejemplo de código:

.hero { border: 5px solid #4ecdc4; }

Tema 4 (Gris)

En tema4.css: tonos grises (#2c3e50), estilo profesional, fuente Georgia.

Ejemplo de código:

.hero { border-left: 8px solid #2c3e50; }
